Addendum

I was unsure if drawing on a canvas using Javascript was what you wanted, so I also added a bubble sort function tied to a button click. For the canvas, I first had to get the id of a div where the canvas was going to be located. Using the width and height of that div, it creates a canvas and places it under the div. I made a particle class that has a few functions that get called with every draw. Update position/speed calculates a new random modifier to add to the speed at every call and updates the position using that speed. Check for collisions checks for collisions with walls and calculates the new trajectory. Update tail will create, push to, and use a position history to make tail pieces. Show will show all of the pieces together with every draw call. The bubble sort function takes in a static array and sorts it at the push of a button. The button updates the html of a p with the sorted array.
